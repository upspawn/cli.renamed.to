# renamed-to

This is an alias package for [`@renamed-to/cli`](https://www.npmjs.com/package/@renamed-to/cli).

## Installation

```bash
npm install -g renamed-to
```

Or with pnpm:

```bash
pnpm add -g renamed-to
```

## Usage

After installation, you can use the `renamed` command:

```bash
renamed auth device
renamed rename file.pdf
renamed extract invoice.pdf --schema invoice
renamed pdf-split multi-doc.pdf --wait
```
